# jryspro-api
 The backend api for koishi-plugin-jryspro  
 koishi-plugin-jryspro的后端api,如果有需要可以自取  

# 部署
## 条件
* [nodejs(v16+)](https://nodejs.org)
* 推荐用[yarn包管理器](https://www.yarnpkg.cn/)

## 快速上手
1、克隆本仓库  
2、在项目根目录安装依赖  
yarn  
```
yarn
```
npm  
```
npm i
```
3、安装完成后，在项目根目录输入以下命令运行本项目  
```
node .
```
或者使用yarn  
```
yarn start
```
npm  
```
npm run start
```

## 修改配置
在源码中修改即可  
如果需要添加一些运势签可以在.csv文件中按照内容中的格式添加  
csv文件不可删除
